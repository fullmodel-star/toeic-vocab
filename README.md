# 💼 多益單字（TOEIC）

一個**離線可用、免安裝、零追蹤**的多益核心單字學習 PWA，為成人／上班族準備多益(TOEIC)常考字彙。沿用「高中 6000」的單字卡引擎，並加上**KK 音標**。

> 純單一 HTML 檔、資料全存本機、不上傳雲端、免註冊、無廣告。

> **狀態**：✅ 已上線　｜　🔵 深藍主題（theme `#1e3a5f`）　｜　📖 **874 字**　｜　🔊 音標＋朗讀　｜　📱 響應式
> 上線網址：<https://fullmodel-star.github.io/toeic-vocab/>　｜　已收錄於[英語學習入口 hub](../14_英語學習入口/)

---

## ✨ 功能
- **874 個多益常考字**：每字含詞性、KK 音標、繁中字義、例句與中譯
- **單字卡**：3D 翻卡、滑動評分、翻面朗讀、SRS 間隔複習（Leitner）
- **單元闖關 + 測驗**：英中雙向／拼字／填空，過關解鎖
- **錯題本、遊戲化 XP、今日任務、深色模式**（同 6000 引擎）

## 🚀 使用
瀏覽器開 `index.html` 即可；安裝成 App 需用上線 https 網址（Android Chrome →「安裝應用程式」，iPhone Safari →「加入主畫面」）。

## 📁 檔案結構
| 檔案 | 說明 |
|---|---|
| `index.html` | 主程式（單檔即整個 App） |
| `manifest.json` / `sw.js` | PWA 設定與離線快取 |
| `icon-*.png` / `apple-touch-icon.png` / `icon.svg` | 圖示 |

## 🔒 隱私
完全離線、不連外部伺服器；學習紀錄只存裝置 `localStorage`（key 有專屬後綴，與其他 App 不衝突）；無帳號、無追蹤、無廣告。
